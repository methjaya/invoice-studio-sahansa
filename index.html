One-time setup — connect your free Supabase database
1
Go to supabase.com → Create a free account → Click New Project. Wait ~2 minutes for it to be ready.
2
In your project, click SQL Editor in the left sidebar. Paste the Setup SQL (from the guide below) and click Run.
3
Go to Project Settings → API. Copy your Project URL and anon / public key and paste them here.
