One-time setup — connect your free Supabase database
1
Go to supabase.com → Create a free account → Click New Project. Wait ~2 min.
2
In your project → SQL Editor → paste and run the supabase-setup.sql file.
3
Go to Project Settings → API Keys → copy your Project URL and anon / public key.
